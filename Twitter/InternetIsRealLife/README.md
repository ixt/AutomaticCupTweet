# TheInternetIsRealLife
A script and cron definition for my automatic tweets of a cup on my windowsill 

## Requires:  
* [t](http://github.com/sferik/t/) (Twitter client for Ruby)  
* fswebcam (Usually in repos)
* imagemagick
* an actual webcam
